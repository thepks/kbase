This application is used to record and share performance guidelines and other useful information.

Use the search function to find related information such as rule of thumb guides, important parameters and SAP Notes.

Basic functions to search, add and manage performance related information.

To add a new item use the manage button and enter details.

To see full details click on the title of an item in the search results.

To edit search first and then press the manage button to enable the edit function.

To delete search first then press the bin.
